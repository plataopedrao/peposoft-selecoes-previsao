# CLAUDE.md — Procedimentos do projeto peposoft

## Git e versionamento

- **Toda alteração feita no projeto deve ser commitada e enviada ao GitHub ao final da tarefa.**
- Repositório: https://github.com/plataopedrao/peposoft-selecoes-previsao
- Branch principal: `main`
- Autor dos commits: `Pedro Oliveira <pedro@woliveira.net>`
- Nunca incluir `Co-Authored-By: Claude` nas mensagens de commit.

### Fluxo padrão após qualquer alteração

```
git add <arquivos alterados>
git commit -m "mensagem descritiva"
git push origin main
```
